# 🌐 Domain Name Generator

A smart and simple Domain Name Generator built with HTML, CSS, and JavaScript.  
It helps you generate creative business and brand domain ideas instantly.

---

## ✨ Features

- 🔤 Keyword-based domain generation
- 🌐 Multiple domain extensions (.com, .net, .io, .dev)
- 🎲 Random unique domain ideas
- 📋 Click to copy domain
- ⚡ Instant results
- 🎨 Clean modern UI
- 💻 Beginner friendly code

---

## 📂 Project Structure

```
domain-name-generator/
│── index.html
│── style.css
│── app.js
│── README.md
```

---

## 🚀 How to Use

1. Open `index.html` in browser
2. Enter a keyword (e.g. tech, shop, ai)
3. Click **Generate Domains**
4. Click any domain to copy it
